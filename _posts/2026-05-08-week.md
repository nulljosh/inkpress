The brief page is a different thing now. Two cases, per-violation breakdowns, sidebar on desktop, the right fonts. The rest of the week: tally marks on the iOS splash, school tracker rebuilt in the tally style, and the Trakt key's unproductive round trip through a server proxy.

## tuesday

Tally iOS redesign. Three commits. Full UI overhaul -- orange replaces system blue everywhere, solid cards replace the blur material, notification read-state now syncs from the server on mount. Landing page rebuilt as a hero splash. Then a simplification pass. Finished with the avatar -- went from node-graph to pixel-art and back to node-graph before landing on the Anthropic dot-proximity style. iOS home now matches the web: payment progress bar, monthly/next-payment stats grid.

Journal got GitHub Actions CI. ship.sh simplified down to the bits that matter.

## wednesday

Brief page on the portfolio got a full redesign. Two cases merged in -- Two Daryls and a charter rights settlement analysis. Rebuilt the layout around them: tabs per case, accordion for the legal grounds, per-violation damages breakdown. Inter Tight headers, JetBrains Mono for citations, paper texture. It reads like a legal tool now instead of a code project page.

## thursday

Brief's desktop layout finally has the right shape. The page had been single-column since it launched -- fine on mobile but flat on a wide monitor. Sticky hero on the left, case panel beside it.

iOS: dark login screen now, tally marks on the splash. They work better as a launch image than a plain logo -- they're part of what the app does, not just a symbol standing in for it. Pre-Calc 12 switched from online self-paced to a spring structured class finishing mid-June, so the school tracker got updated.

## apps

Grapher landed -- Desmos-style graphing calculator, no external dependencies. PIN auth moved per-user, stored in an encrypted Blob profile instead of shared state. Messages sync was out of step with report status and account info -- fixed all three together.

## nulljosh.github.io

The Trakt API key was hardcoded in client JavaScript. Pulled it out and wrote a serverless proxy at `/api/trakt`. Then Cloudflare blocked the server IP and the stat went blank. Restored it client-side. Net: the key is slightly more exposed but the stat actually works. Will revisit.

Projects trimmed to four shipped -- fuse, bcgd, nimble, nyc, wiretext removed from the list. Tighter.

## saturday

School dashboard caught up to the rest -- orange accent, solid cards, the same weight as tally. It was the only app in the suite still on system blue. A&P 12 done and hidden. Pre-Calc blended reads correctly.

The Trakt proxy that went in early in the week was out by Saturday. Three commits to get back to where we started. Net zero on that one, minus the hours. The brief redesign was a proper afternoon. The iOS splash was fast -- tally marks make more sense as a launch image than a plain logo.
