![May 15](/assets/2026-05-15-week.svg)

The brief went native. A full Swift app in a week -- dark launch screen, tabs, damage tables, Wang v. AG Canada in the comparables. The web version got the same treatment: global comparables, father's witness statement, limitation defense, Ward framework, per-head stack. It's a real tool now.

## sunday

Tally hit v3.5.0. Message read state now syncs cross-platform -- iOS reads what web marked, web reads what iOS marked. The MessagesView got a full redesign. Avatar disk cache added so it stops refetching on every render. Three commits to land stable message IDs before the read state could work at all.

## monday

Dose got a vibe pass. Plus Jakarta Sans headings, warm dark palette -- `#161514` base, `#edebe8` text. Auto dark/light mode and an iOS build fix. Nimble picked up natural language query improvements and version display in macOS Settings.

## tuesday

Life integrated 2019-2024 personal history across web and iOS. That data was always in my head; now it's in the app. Langley Hospital corrected (not Lions Gate -- different hospital, different incident). Fuse removed from apps, claude-usage dropped, roost and spark vision expanded in the README.

## wednesday -- thursday

Brief iOS v1.0.0. Built from scratch in Swift: three tabs (overview, grounds, money), comparable cases table, risk color coding, a scales icon, dark launch screen. Fixed Swift 6 `@Observable` issues, ShapeStyle extension, `UIRequiresFullScreen`. Wang v. AG Canada landed in the native comparables table the same day it went into the web version.

Web brief kept moving in parallel: global comparable awards added, father's witness statement with legal annotations, risks and evidence gaps section, lawyers section, wrist injury documented, draft email template, limitation defense analysis integrated, Ward framework for general damages, per-head stack for household claims. Roadmap for v3 multi-user auth drafted.

## friday

Applied to Capilano University for the paralegal diploma. Suiting up.
