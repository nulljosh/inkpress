Week opened with the Dose and Lingo renames landing for real, then turned into a widget-extension sweep and a pile of roadmap imports from old PDF notes.

## tuesday

Dose renamed to Healstack and fixed a duplicate build output bug in xcodegen. Epiphany marked App Store version 2.5.1 as submitted and refreshed its architecture diagram. Nulljosh.github.io swapped Lingo for Parlay in the project list and fixed stale purple placeholder icons on Echo and Parlay. Quotable picked up a genre filter, share-score button, macOS scaffold, and later a custom subdomain plus light/dark mode and portfolio vibe tokens.

## wednesday

Widget extensions got wired into project.yml across epiphany, spark, and talli, with App Group entitlements added. Spark fixed Swift 6 concurrency errors in its widget target. Echo, epiphany, spark, talli, and parlay all got a What's New popup and imported their old roadmap PDFs (icons-bugs, epiphany-notes, Spark, Talli, Lingo, Echo web) into tracked TODOs, then cleaned out anything already checked off. Nimble added an iOS companion target. Nyc added an iOS companion target with touch controls and wired manual colonist move commands. Dotfiles added a combined `/audit` command (security-sweep + code-review).

Later in the day Epiphany ran an iOS 2.5.2 pass: capsule search bar with a clear button on the map, a location button that cycles three zoom levels, brokerage sync on portfolio refresh, and build 7 uploaded after chasing a widget Info.plist rejection. Parlay got renamed again, this time to Lexly, to match the App Store Connect record. Spark got its Mac App Store Connect record created after fixing a widget embed error, and Talli patched a Swift 6 concurrency error in its widget. Quotable shipped v1.3.0 with 143 quotes, a settings panel with sound effects, and a cleanup pass that stripped gradients, purple accents, and emojis from the UI. The night closed with a design token sync: echo, grapher, spark, and wiretext now pull their palette from the portfolio's tokens.css, dropping the old brown and cream leftovers. Echo adopted the same portfolio vibe tokens and killed its purple. Talli shipped v3.5.3: blue accent, report banner and avatar data fixes, a messages refresh, and the widget building clean under Swift 6. The portfolio grew a 1930s rubber-hose theme, tokens-30s.css with a brush mascot, with readability and rollout items queued on the roadmap.

Evening update: Epiphany's SnapTrade phantom holdings bug got fixed by deduping accounts by id in listAccounts and keying holdings merge on account id, resolving a post-launch blocker. The fix also tolerates empty response bodies from SnapTrade disconnects. README updated with autopilot broker decision: IBKR for stocks, WS read-only, Kraken parked. What's New modal removed from web (iOS-only feature).

## thursday

Early-morning Lexly cleanup: branding fixes across the repo, a lingo-to-lexly redirect, generated and uploaded avatars, and a sweep of leftover Lingo/Parlay names from the docs. The portfolio project list now points at Lexly with the new URL.

Later, a new project called root got scaffolded: an etymology app with a Wiktionary parse pipeline, a SwiftUI multiplatform stub, and a web stub. The parser already handles inh/der/bor templates and root templates, and later refined to handle inh+/der+/bor+ variants. Spark hit Vercel's 12-function Hobby cap and got its auth routes consolidated into a single function with a rewrite after the dynamic route refused to match. Wiretext dropped its unused Fraunces and DM Sans font links now that it pulls tokens from the portfolio. Echo adopted the portfolio's vibe tokens. NYC polished its HUD and tutorial hit-testing, refreshed building sprites with detail and Retina density variants, expanded controls reference, and removed the autoplay directive engine to put the game fully under player control. Talli verified v3.5.3 against pending checklist items.

Evening: Lexly sign-in hang verified fixed and live, avatar step removed from signup flow, stale lingo/ clone deleted after rescuing icon fixes, macOS icon rebuilt on Apple's icon grid (824px rounded rect, all scale sizes). All App Store metadata and screenshots filled via asc for both iOS and macOS, with submission blocked only on 2FA availability and App Privacy policy completion. Created /progress skill to track whole-codebase time/cost estimates: ~67 open items ≈ 155 hours across all repos. Lexly's iOS and macOS app icons were also regenerated so the artwork fills the canvas instead of sitting at roughly 60% size.

Later night: Spark Mac notched its first release build (1.0.0 build 1), hit the provisioning wall, fixed it with xcodebuild -allowProvisioningUpdates, archived and exported the build, then uploaded to App Store Connect. Lexly iOS and macOS builds both confirmed valid in the App Store build pipeline. Books project added two new summaries (Apple: The First 50 Years and Exercise Science For Dummies) from photographed chapters, with images cleared from iCloud and badges added to index.html.

Nightly investigation: Talli Xcode Cloud builds were failing on 50+ jobs due to a missing iOS Distribution certificate. Diagnosed via asc-cli and created the missing cert on App Store Connect using `asc certificates create --certificate-type IOS_DISTRIBUTION`. Reviewed the monthly-report submission code path (src/scraper.js), confirmed it completes a real end-to-end filing to BC Self-Serve with all form fields intact, but flagged weak success-detection logic that should be hardened with stronger assertions before the next filing window (days 1-5 of the month).

## friday

Lexly's macOS app icons were regenerated to fill the canvas at full size; Nulljosh.github.io added UVic timeline entry for returning to school to pursue Computer Science and retaking Pre-Calc 12; Talli tightened monthly-report submit success detection to avoid false positives on form re-renders. Later, Epiphany locked scan-local statements endpoint to dev-only and fixed iOS splash screen using old grey wireframe icon. Talli added Mac widget provider Swift 6 concurrency fixes and appearance settings. Echo regenerated App Store screenshots, updated What's New copy, added Voice Memos import with transcription progress bar and animation polish. Brief fixed Supabase env-var crash, dismissed What's New under sheet, and adopted portfolio design tokens on web. NYC pushed 6 commits through the App Store submission pipeline: registered iOS bundle ID with provisioning profile, generated a distribution certificate for iOS IPA export, metadata-filled both platforms, fixed 5 of 7 macOS blockers via ASC API (category, review contact, encryption, content rights, age rating), set free pricing, uploaded the first iOS build to ASC, and confirmed both macOS 1.0 and iOS 1.0.0 validate cleanly (one remaining macOS validation error pending). Also installed Docker Desktop and set up Shannon (KeygraphHQ's AI pentester) as a new /pentest skill with a Claude Code OAuth token; first test run against talli's dev server failed silently and needs debugging before Shannon is production-ready.

Evening: Talli fixed Xcode Cloud signing failures by renaming App Group to match the bundle ID namespace (was causing 50+ job failures).

Nightly: Spark fixed invisible dark-mode text by defining --text/--bg/--text3 locally instead of relying on cross-origin @import timing. NYC hit an iOS App Store rejection, fixed bundle app icon assets, iPad orientations, and launch screen, bumped the iOS build to 3, and documented the upload failure in the roadmap.
