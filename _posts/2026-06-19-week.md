The apps monorepo came apart this week. Everything went back to standalone repos, with the small experiments consolidated into one labs repo instead of scattered on their own.

## monday

Tally synced its i18n locales, adding RDSP and CDB translation keys. Echo got a redesigned app icon.

## tuesday

Brief came back fully restored across iOS, macOS, and web. Dose got cleaned up as a standalone project, and nimble, nyc, grapher, and wiretext all merged into a single labs repo instead of living as separate small repos.

## wednesday

Tally cleaned up the school dashboard, fixed a Capilano typo, and marked Pre-Calc 12 done. Echo's docs got simplified ahead of its App Store push, and dose dropped npm test from its GitHub Pages workflow.

## thursday

The big day. The portfolio site got a full homepage redesign with featured project cards, a live strip, and a timeline, plus a palette swap to blue/black/white/gray. Repo links there now point at the consolidated labs repo. Dotfiles fixed Project Map SVGs that weren't rendering. Brief replaced a pin bypass with an opaque scrape token and fixed stale docs. Tally generalized its PWD/RDSP/CDB endpoints into one data-driven schema instead of three near-duplicates. Echo's web app came back online at echo.heyitsmejosh.com. Journal got re-separated into its own repo again.

## friday

Epiphany's iOS Markets view got a polish pass: sparkline ticker on the full drawer, an ellipsis filter menu, flush list rows, company names instead of just tickers. Autopilot's live mode got capped to a BTC probe instead of running against the full stock watchlist.

## saturday

Echo's Release build got fixed, an invalid suppressTokens assignment was breaking it. Epiphany lost two dead files (simBenchmark.js, stocks-debug.js) and fixed gaps in Langley crime data, flight details, and event/place metadata. Tally added avatar persistence, dropped a misleading $/hr accrual figure, and shipped its App Store privacy manifest with automated screenshots.

## sunday

Spark's full source came back from a Vercel deployment-files API recovery, web, api, iOS, macOS, watchOS, widgets, daemon, migrations, and tests all restored. Lingo's frontend got recovered the same way from its live deployment. Dose confirmed its account-recovery email path against the real Supabase auth.users table. Brief added first-brief authoring and finalized multi-user support. Talli fixed macOS sign-in to surface the real BC Self-Serve login error instead of a generic session-expired message. Echo made its settings icon more visible and tuned down the transcript scroll feel. Epiphany fixed a flights fallback timeout, an NYC IP-geo bug, and disabled a broken Trade tab. The portfolio site got a copy rewrite with expanded project bullets. Dotfiles moved the Cloudflare token out of tracked config into an untracked secrets file.

June is hot this year, 30C most days, average for the month but still sweltering when it sticks around. Development across the apps is going well and the workflow itself keeps improving, CI, code review, the whole loop is getting smoother and it has been genuinely fun. Got accepted into Capilano University for legal studies. The plan is law first, 3-4 years practicing while keeping math and code going in the background, then a UVic comp sci BSc somewhere around 30, mirroring finance the whole way through, law then comp-sci then finance. Three projects are shipped now and the focus shifts to polishing and monetizing them instead of just building. Still chipping at code box gaps and a few lost steps. Father's Day tomorrow. Next up is more code review, more tests, math studying, and styling passes.
